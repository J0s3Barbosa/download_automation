# About the project 
```bach
- download automation
- it downloads the pdfs in the website and stores them in the folder for each subject 
```
# Stack
```bach
    - python
    - playwright
        - https://playwright.dev/python/docs/intro
```

# Instalation

```bach
    pip install pytest-playwright python-dotenv
    playwright install
```

# Usage
```bach
    - install dependencies

    - run main file
```
